# PortScan
A simple Python port scanner tool to check open and closed ports on a target IP...

# 🔧 Port Scanner Tool / Port Skanner Aləti

Bu Python ilə yazılmış sadə port skanner alətidir. Hədəf IP ünvanında portların açıq, bağlı və ya rədd edilmiş olduğunu yoxlayır və nəticələri JSON formatında saxlayır.  

This is a simple Python port scanner tool. It scans a target IP for open, closed, or connection-refused ports and saves the results in JSON format.

---

## ⚡ Xüsusiyyətlər / Features

- Ən çox istifadə olunan portları skan edir: 21, 22, 23, 25, 53, 67, 68, 80, 110, 135, 139, 145, 149, 443, 445, 3389, 8080  
- Açıq, bağlı və rədd edilmiş portları aşkar edir  
- Nəticələr JSON formatında saxlanılır  
- Terminal çıxışı rənglə kodlanır ki, nəticələri asanlıqla oxuya biləsiniz  

Scans common ports: 21, 22, 23, 25, 53, 67, 68, 80, 110, 135, 139, 145, 149, 443, 445, 3389, 8080  
Detects open, closed, or connection-refused ports  
Results are saved in JSON format  
Color-coded terminal output for easy identification
